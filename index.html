<x-geolocation>

Geolocation API Wrapper Element.

Star Follow @1000ch

Install

# via npm
$ npm install x-geolocation

Usage

Import x-geolocation.js and register.

<script type="module">
  import XGeolocation from './x-geolocation.js';
  customElements.define('x-geolocation', XGeolocation);
</script>

Put <x-geolocation> tag.

<x-geolocation></x-geolocation>
